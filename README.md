# Security-Operations-Center-SOC-detection-lab

A hands-on Security Operations Center (SOC) detection lab built using Splunk Enterprise, Sysmon, and Kali Linux to simulate attacker activity and detect it through log analysis and threat detection rules.  This project demonstrates realistic SOC analyst workflows, including log ingestion, attack simulation, and detection engineering.
